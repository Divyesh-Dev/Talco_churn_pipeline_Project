Step 1 — Install dependencies

pip install pytest pytest-cov jupyter

Step 2 — Run notebooks

Open all 4 notebooks in browser

jupyter notebook notebooks/

Open a specific notebook

jupyter notebook notebooks/01_data_understanding.ipynb

Once open — run all cells in order

Menu → Cell → Run All or press Shift + Enter cell by cell

Step 3 — Run unit tests

Run all 35 tests with details

python -m pytest tests/ -v

Run with coverage report (shows % of code tested)

python -m pytest tests/ -v --tb=short --cov=src

Run only one test class (e.g. cleaning tests)

python -m pytest tests/ -v -k "TestCleaning"

What each notebook does

01_data_understanding Load raw CSV, check shape, data types, missing values, churn distribution charts, correlation matrix. Run this first.
02_data_cleaning Fix TotalCharges, impute with median, remove duplicates, strip whitespace. Shows before vs after comparison table.
03_feature_engineering Binary encode columns, one-hot encode categories, create tenure_group, RevenueCategory, AnnualRevenue, LoyaltyScore. Charts for each feature vs churn.
04_visualization All business charts — revenue at risk, churn across every feature, ML model performance, offer distribution, final KPI summary. Needs pipeline run first.

What each test class covers (9 classes, 35 tests)

TestExtract (5 tests)

Column presence, uniqueness, correct shape

TestCleaning (6 tests)

TotalCharges fix, null imputation, duplicates, whitespace

TestTransform (10 tests)

Churn encoding, OHE, Contract/Internet/Payment preserved

TestFeatureEngineering (7 tests)

New columns, AnnualRevenue = 12x monthly, LoyaltyScore in [0,1]

TestValidation (3 tests)

Passes on clean data, fails on nulls and bad dtype

TestPredict (3 tests) — needs model

Probability in [0,1], risk label valid, output is dict. Auto-skips if model not trained yet.

TestRecommend (7 tests)

Correct offer per customer type, all required fields present

TestAuth (4 tests)

Password hash/verify, users.json exists, admin user, add/delete user

TestConfig (3 tests)

All imports work, paths are strings, tenure bins sorted
